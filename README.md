# 🚀 Project Overview

**BlackWall Security Suite** is a full-stack cybersecurity project designed for:

- 🛡️ Network traffic visibility  
- 🔍 Host & service analysis  
- 📊 System performance monitoring  
- 🚨 Suspicious behavior alerts  
- 📝 Automated security reporting  

Built for:

- Cybersecurity students  
- Blue team learners  
- SOC analysts  
- Academic mini / final year projects  
- Professional portfolios

# 📦 Installation

## 1️⃣ Clone Repository
```bash
git clone https://github.com/kdkavi/ddos-hacks/blackwall.git
```
```
cd ddos-hacks
```

## 2️⃣ Install Requirements
```bash
pip install -r requirements.txt
```

If pip not working:
```bash
python -m pip install -r requirements.txt
```

## 3️⃣ Run Backend (CLI Engine)
```bash
python blackwall.py
```

## 4️⃣ Run Frontend (Dashboard)
```bash
cd web
start index.html
```
